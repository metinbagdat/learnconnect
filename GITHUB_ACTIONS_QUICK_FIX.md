# GitHub Actions Quick Fix - DATABASE_URL Error

## ❌ Error
```
Error Please provide required params for Postgres driver: [x] url: ''
```

## 🔍 Root Cause
The workflow needs **NEON_API_KEY** and **NEON_PROJECT_ID** to create a Neon branch and get the DATABASE_URL.

## ✅ Solution: Add GitHub Secrets/Variables

### Step 1: Get Your Neon Credentials

1. Go to https://console.neon.tech/
2. Log in to your account

#### Get NEON_PROJECT_ID:
1. Select your project
2. Go to **Settings** → **General**
3. Copy the **Project ID** (looks like: `ep-xxxxx-xxxxx`)

#### Get NEON_API_KEY:
1. In Neon Console, click your profile → **Account Settings**
2. Go to **Developer Settings**
3. Click **Create API Key**
4. Give it a name (e.g., "GitHub Actions")
5. Copy the API key (starts with `neon_`)

### Step 2: Add to GitHub

1. Go to: https://github.com/metinbagdat/learnconnect-/settings/secrets/actions

#### Add Secret (NEON_API_KEY):
- Click **"New repository secret"**
- **Name:** `NEON_API_KEY`
- **Value:** Your Neon API key (from Step 1)
- Click **"Add secret"**

#### Add Variable (NEON_PROJECT_ID):
- Click **"Variables"** tab
- Click **"New repository variable"**
- **Name:** `NEON_PROJECT_ID`
- **Value:** Your Neon Project ID (from Step 1)
- Click **"Add variable"**

### Step 3: Test the Workflow

1. Go to your PR
2. Push a new commit OR close and reopen the PR
3. Check GitHub Actions tab
4. The workflow should now:
   - ✅ Create a Neon branch
   - ✅ Get the DATABASE_URL
   - ✅ Run migrations successfully

## ⚠️ Important Notes

- These are **DIFFERENT** from Vercel environment variables
- GitHub Actions needs Neon credentials to create preview database branches
- The DATABASE_URL is automatically generated by Neon when the branch is created
